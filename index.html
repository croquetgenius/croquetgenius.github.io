Part-of-speech tagger for inflected ancient Greek

Capstone Project for General Assembly Data Science Immersive

In this project I leveraged Tensorflow to create a POS tagger, then deployed this to the web.

Tools used:
  • Python
  • pyconll
  • Pandas
  • Tensorflow
  • NLTK
  • Flask
  • HTML
  • CSS
  • Heroku

Ancient Greek has always been the lesser-taught of the two classical languages (ancient Greek and Latin). Most of the classical language tools, apps, and programs are built specifically for Latin or with ancient Greek as a kind of 'stretch-goal'. In the past decade, NLP has made great strides (digital assistants like Alexa and Siri, bots, Google translate, etc.) and there are a ton of tools out there for processing and modeling modern languages. However, once again, ancient Greek gets none of the NLP love.

For this project I used pre-tagged data from Universal Dependencies, which was then cleaned, tokenized, and vectorized into character-level matrices using Keras with a Tensorflow back-end. Then I trained multiple CNNs on the data, concatenating their outputs before using a Dense layer to predict the POS tag. I then deployed my model to the web using Flask, Gunicorn, and Heroku. The web app takes user input in the form of a string and returns the predicted POS tags.

COVID-19 Twitter Sentiment Analysis

Client/Group Project for General Assembly Data Science Immersive

For this group project, my team analyzed the sentiments of tweets from across the U.S. to see if states' differing responses to COVID-19 affected Twitter users' attitudes. We used:

Tools used:
  • Python
  • GetOldTweets3
  • spaCy
  • CountVectorizer
  • VaderSentiment
  • Matplotlib
We worked with GetOldTweets3 to get tweets from key cities in states with the most severe and least severe lockdown measures during most of April 2020. As a control, we also pulled tweets from the same cities and dates from 2019. We ended up with more than 700,000 tweets from twenty cities. We then used spaCy to process the text data and VaderSentiment for sentiment analyziz. Finally, we plotted the sentiments to identify trends and connections. However, tweet sentiments didn't seem to be affected by differences in statewide policies responding to COVID-19, nor did sentiments change much between 2019 and 2020.
